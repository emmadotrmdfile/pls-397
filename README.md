## README

These are two of my first “serious” R projects from 2023.

The "CCC Analysis" file uses data from the *Crowd Counting Consortium,* where I focus specifically on reproductive rights and analyze protest activity related to those issues.

The "TRIP Analysis" file draws on data from the *Trans Rights Indicator Project,* examining the legal protections transgender individuals face, and the gaps in those protections, across countries worldwide.
